https://leonardo-portfolio.herokuapp.com/

## Run Locally

Clone the project

```bash
  git clone https://github.com/leonardo-nunez/fullstack-chat-leo
```

Go to the project directory

```bash
  cd fullstack-chat-leo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
